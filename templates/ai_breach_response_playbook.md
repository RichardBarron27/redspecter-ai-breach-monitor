# AI Breach Response Playbook

If a breach alert fires:

---

## 1. Confirm & Triage
- Verify alert
- Check logs for context
- Classify severity

---

## 2. Contain
- Stop affected process
- Disable risky prompts or tool
- Rotate exposed secrets

---

## 3. Assess Impact
- Scope affected logs/time range
- Determine exposure path

---

## 4. Recover
- Fix misconfigurations
- Update patterns / controls

---

## 5. Learn & Close
- Document using incident log template
- Improve policy and user awareness
